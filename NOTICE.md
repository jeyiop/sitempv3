# NOTICE RAPIDE (pour toute l'équipe)

## Objectif
Avoir une base propre, compréhensible et facile à faire évoluer sans casser le design.

## Ce qu’il faut retenir
- Le haut de page (zone slider) reste clair/blanc et homogène.
- Les sections de contenu peuvent garder leur identité visuelle.
- Les images doivent rester nettes et non rognées.
- Les changements se font dans la copie Codex uniquement.

## Process simple quand vous ajoutez des images
1. Déposer vos images dans `selecta`.
2. Les copier dans les bons sous-dossiers `public/image/selecta/...`.
3. Vérifier le rendu sur `http://localhost:3010`.
4. Hard refresh (`Cmd+Shift+R`) si nécessaire.

## Process simple avant gros changement
1. Faire une sauvegarde dans `archives`.
2. Modifier.
3. Vérifier desktop + mobile.
4. Documenter en 2-3 lignes ce qui a changé.

## Références utiles
- Projet: `V0_Site_Public_copy`
- Dépôt média: `selecta`
- Sauvegardes: `archives`

## Vérification finale (checklist)
- [ ] Images nettes
- [ ] Pas de rognage non voulu
- [ ] Alignements cohérents
- [ ] Textes lisibles et professionnels
- [ ] Liens importants fonctionnels (`/simulateur`, `/devis`, `/contact`)
